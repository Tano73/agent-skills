# Costi Standard Figure Professionali (2025)

Fonte: `estimator-import-template.xlsx` → sheet **Costi Std**  
Valori espressi in **€/giorno** (costo standard interno).

---

## Guida ai codici

Il codice è composto da 4 caratteri: `<PROFILO><LIVELLO><GRADO>`

**Prefisso profilo:**
| Prefisso | Famiglia professionale |
|----------|----------------------|
| TD | TECHNICAL / DEVELOPMENT (sviluppatori, architect, lead) |
| CO | CONSULTING (BA, PM, consulenti) |
| CL | CLOUD (cloud & infra engineers) |
| AG | AGILE (scrum master, agile coach) |
| AI | ANALYTICS & AI (data engineer, data scientist) |
| CY | CYBERSECURITY |
| OP | OPERATIONS |
| IT | INFORMATION TECHNOLOGY (generalisti IT) |
| AR | AUDIT & RISK |
| FIN | FINANCE |
| GAN | GENERAL & ADMIN SERVICES |
| GMN | GENERAL MANAGEMENT |
| SC | STRATEGY & CORPORATE DEVELOPMENT |
| SE | SECURITY |

**Livello (numero):**  
0 = Trainee/Entry · 1 = Junior · 2 = Mid · 3 = Senior · 4 = Lead/Expert · 5 = Principal/Manager

**Grado (ultima lettera):**  
D = Director/Arch (costo più alto) · E = Standard Senior · F = Mid · G = Junior · H = variante specialistica · Z = Trainee
